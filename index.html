Kiszolgáló alapfogalmak

DHCP (Dynamic Host Configuration Protocol)

A DHCP (Dynamic Host Configuration Protocol) egy protokoll, amely gyors, automatikus és központi kezelést biztosít egy hálózaton belüli IP-címek szétosztásához[^.^].

DNS (Domain Name System)

A DNS (Domain Name System) a webes domain neveket IP-címekké alakítja, amelyeket a böngészők használnak az internetes oldalak betöltéséhez[^.^].

IIS (Internet Information Services)

Az IIS (Internet Information Services) egy web szerver szoftvercsomag, amelyet a Windows Server számára terveztek. Weboldalak és egyéb tartalmak weben történő tárolására használják[^.^].

AD

Az AD (Active Directory) egy Microsoft technológia, amelyet a felhasználói fiókok és a hozzájuk tartozó erőforrások kezelésére használnak a hálózaton.

Fájlkiszolgáló

A fájlkiszolgáló olyan szerver, amely a hálózaton keresztül tárolt fájlokhoz biztosít hozzáférést a felhasználók számára.
